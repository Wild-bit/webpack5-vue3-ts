# webpack-vue3-ts

## 项目简介

项目基于 webpack 5 实现了 vue3 + ts 的项目搭建，采用了 vue3 最新相关技术栈 vue-router4 以及 pinia，具体搭建过程可见我的博客https://juejin.cn/post/7155138156352765960


## 项目目录

```
. 
├── public
| └── index.html
├── src 
| └── pages
| └── router
| └── store
| └── components
| └── App.vue
| └── index.ts
└── pnpm-lock.yml

└── package.json
└── tsconfig.json
└── webpack.config.ts
```
